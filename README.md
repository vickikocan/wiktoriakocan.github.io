# wiktoriakocan.github.io

Personal website for Wiktoria Kocan

## Running the site locally

Super simple to run the site locally, I use [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) to run it + [Live Sass Compiler](https://marketplace.visualstudio.com/items?itemName=ritwickdey.live-sass) to compile SCSS on the fly.
